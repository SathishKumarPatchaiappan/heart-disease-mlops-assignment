# Required Screenshot Checklist

Capture screenshots using your own machine and GitHub account.

1. EDA notebook outputs: missing values, class balance, histograms, heatmap, feature relationship.
2. Model comparison table and terminal training output.
3. MLflow experiment list, run metrics, parameters, artifacts, confusion matrix, ROC curve, saved model.
4. Pytest passing output.
5. GitHub Actions successful workflow with lint, tests, training, artifact upload, and Docker build.
6. Docker image build and running container.
7. FastAPI Swagger `/docs`, `/health`, and successful `/predict` response.
8. Kubernetes `kubectl get pods`, `kubectl get services`, and API response through Minikube.
9. Prometheus target status and metrics query.
10. Grafana dashboard showing request count and latency.
